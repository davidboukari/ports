# ports

* See https://guide.macports.org/ 

```bash
port help selfupdate
port list
port search <package>
port info <package>
port deps apache2 
port deps apache2 +openldap
port uninstall <portname>
port contents xorg-renderproto
port installed

# from which package come
port provides /opt/local/etc/irssi.conf
```
